使い方（How to）

SpeakNoteの使い方

英語例文を登録して、再生・録音・シャドーイングまでを1つの流れで練習できる学習アプリです。
まずは短い例文から始め、慣れたら対話文や60秒スピーチへ広げるのがおすすめです。

できること

  • 場面ごとに英語例文と和訳を保存
  • 英文の音声再生と速度調整
  • 録音して自分の発話を確認
  • シャドーイングで反復練習
  • 検索・タグ・カテゴリで整理

おすすめの使い方

  • 短い例文を毎日3〜5本練習する
  • 同じ英文を「聞く→読む→話す」で3回回す
  • 意見問題は理由を2つ入れる形で覚える
  • 対話文はW: / M: の流れごとに練習する

まずはこの流れ

入力する → 保存する → 一覧で開く → 再生する → 録音する → シャドーイングで反復する
1
例文を用意する

練習したい英文を1つ選びます。自分で作っても、AIで下書きを作っても大丈夫です。最初は短くて言いやすい文から始めると続けやすくなります。

AIに頼む例
  • 「自己紹介の英語スピーチを45秒以内で作って。日本語訳もつけて。」
  • 「TOEIC Speakingの意見問題で使いやすい英文を、理由2つ入りで作って。」
  • 「受付で道を聞く場面の短い対話文を、W: と M: 形式で作って。」
  • 「写真描写で使いやすい基本表現を5つ、英語と日本語で作って。」
2
登録する

「登録」タブで、場面・英文・和訳・カテゴリ・タグを入力して保存します。必須なのは 場面 と 英文 です。

  • 場面：いつ使う文かを短く書く
  • 英文：実際に話す文を入れる
  • 和訳：意味確認用。あとから追加も可能
  • カテゴリ：自己紹介、意見、写真描写など
  • タグ：依頼、会議、旅行、面接 など自由入力
例） 場面: 会議で自己紹介するとき 英文: Hello, everyone. My name is Alex Tanaka... 和訳: 皆さん、こんにちは。私の名前はアレックス田中です... カテゴリ: 自己紹介
3
一覧から開いて練習する

「一覧」タブでは、保存した例文を検索・絞り込みできます。カードをタップすると詳細画面が開き、再生や録音ができます。

  • ▶︎ 再生：英文を読み上げ
  • − / ＋ 速度：自分のレベルに合わせて調整
  • ● REC：自分の声を録音して確認
  • 編集：文の修正やタグの追加
  • 削除：不要になった例文を削除
4
シャドーイングで仕上げる

「シャドー」タブでは、カテゴリやタグで対象を絞って連続再生できます。英文の単語がハイライトされるので、音を追いかけながら発話しやすくなります。

  • 最初は0.8〜1.0倍で開始
  • 慣れたら1.1〜1.2倍で挑戦
  • 対話文は W: と M: の切り替わりを意識
  • 一時停止して言い直し、再開して繰り返す

TOEIC Speaking向けのコツ

自己紹介・説明問題： 最初に結論、そのあと補足を1〜2文足すと話しやすくなります。
意見問題： 「I think ...」のあとに理由を2つ入れる形にすると安定します。
写真描写： 人・動作・場所・雰囲気の順で見ると、言う内容が整理しやすいです。
対話文： 短い返答を素早く言えるように、受付・予定変更・依頼などの定番場面を反復すると効果的です。

データ管理

Export

保存した例文をJSONでバックアップできます。端末変更前や、大きな編集前に保存しておくと安心です。

Import

以前のバックアップを読み込めます。既存データを入れ替えるか、追加するかを選べます。

個別削除

詳細画面の「削除」ボタンから、その例文だけ削除できます。

全削除

設定画面の「ﾘｾｯﾄ」で全例文を削除できます。元に戻せないので実行前にExportがおすすめです。

困ったとき

音が出ない： 設定でVoiceを選び直して保存し、端末の音量や消音設定も確認してください。
再生が止まる： もう一度再生ボタンを押してください。長文は短めの文に分けると安定しやすいです。
ハイライトが追いつかない： 速度を少し下げると見やすくなります。
何から始めるか迷う： まずは自己紹介、意見、写真描写の3カテゴリから始めるのがおすすめです。
